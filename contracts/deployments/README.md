# Deployment Addresses

This directory contains deployment information for FoundersNetMarket contract across different networks.

## Files

- `amoy.json` - Polygon Amoy testnet deployment
- `polygon.json` - Polygon PoS mainnet deployment

## Format

Each JSON file contains:

```json
{
  "network": "amoy",
  "chainId": 80002,
  "marketContract": "0x...",
  "usdcToken": "0x...",
  "deployer": "0x...",
  "deployedAt": "2025-12-16T10:00:00.000Z",
  "transactionHash": "0x...",
  "lastUpdated": "2025-12-16T10:00:00.000Z"
}
```

## Usage

These files are automatically generated by the deployment script:

```bash
npx hardhat deploy --network amoy
```

Frontend and backend can import these files to get contract addresses:

```typescript
import amoyDeployment from './deployments/amoy.json';
const contractAddress = amoyDeployment.marketContract;
```

## Security

These files contain **public** addresses only. No private keys or sensitive data.

Safe to commit to version control.
